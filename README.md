## How to Run

1. **Install dependencies:**
   ```powershell
   npm install
   ```

2. **Start the server:**
   ```powershell
   npm start
   # or
   node server.js
   ```

3. **Open your browser:**
   - Navigate to `http://localhost:3000`
   - Your Leadership Dashboard will load
